Per l'esecuzione del progetto è necessario VisiBroker for Java 4 ed una macchina virtuale Java (versione 1.3).

Installare il VisiBroker ed assicurarsi che la directory bin sia nel path.

Se si vuole anche compilare il codice è necessario che le librerie fornite dal VisiBroker siano nel classpath.

Quindi eseguire il file go.bat che avvia l'osagent in Naming Service  e l'Interface Repository.

A questo punto è possibile registrare i servizi desiderati attraverso i file registraBank.bat, registraProva.bat, registraPrint.bat e registraDB.bat.

Avviare anche i server (dei servizi che si registrano) attraverso i file BankServer.bat, ProvaServer.bat, PrintServer.bat. Ci sono poi 3 server per il servizio di database: DBServer1.bat DBServer2.bat e DBServer3.bat. Naturalmente se ne possono avviare anche altri specificando il nome del server e la tabella da usare che si preferiscono.

Il client può essere avviato in un qualunque momento dopo il go.

Per deregistrare un servizio sono disponibili i file deregBank.bat, deregProva.bat, deregPrint.bat e deregDB.bat.
